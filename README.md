# webpack-VueRouter-Template

## Description
> Example using Vue.js, Vue-router and html-loader with webpack.

> You can use this files as a minimum template of your SPA project with Vue.js and Vue-router.

---

## Requirement
- Node.js 4+

## Setup

    npm install -g gulp
    npm install

And then:

    $ gulp
